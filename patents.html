Patents

I'm slowly going through each year of patents, and as I mount each XML file (yeah I said mount), I search through the title, abstract, and description of each patent for the phrase "application programming interface". If a patent contains these words, I pull out and include in an API, and resulting JSON file stored on this site for exploration.

It takes time, compute, storage, and bandwidth resources to process these, and as I add new files, I will publish the resulting information here. If you'd like access to the resulting API, let me know.
